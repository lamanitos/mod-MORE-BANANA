#MORE BANANA!!!

The mod adds several bananas, namely:
1. Yellow Banana: a regular banana that can be slippery.
2. Blue Banana: gives the Mandrill x2 speed (considering I added speed damage to the robot, this could be a problem). However, it has a debuff, making it easier for the robot to hit the yellow banana.
3. Red Banana: gives the Mandrill a temporary berserk.
4. Classified.
